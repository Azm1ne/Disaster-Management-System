# Frontend

React + Vite + TypeScript single-page app for the Disaster Management System.

See the [root README](../README.md) for the full stack, quick start, and test commands.

## Scripts

```bash
npm run dev      # Vite dev server (proxies /api to localhost:8080)
npm run build    # type-check + production build
npm test         # Vitest
npm run lint     # oxlint
```
